# StudyBuddy (Web)

A productivity web application built with React designed to help students manage tasks, improve focus, and organize study sessions.

---

## Features

- Pomodoro Timer for focused study sessions
- Task management system
- Eisenhower Matrix for prioritization
- Responsive UI for all devices
- State-based task tracking
- Clean and minimal dashboard

---

## Tech Stack

- React
- JavaScript (ES6+)
- Tailwind CSS / Styled Components 
- Context API / State Management

---

## Project Structure

- `components/` – UI components
- `pages/` – Main pages
- `context/` – State management
- `utils/` – Helper functions

---

## Highlights

- Focused productivity design
- Time management tools
- Clean React architecture
